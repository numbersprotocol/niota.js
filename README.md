# niota.js

Install

```
$ npm i @numbersprotocol/niota
```

Run example

1. Create an example transaction 

    ```
    $ mkdir test && cd test
    $ npm i @numbersprotocol/niota
    $ cp ../example/* .
    $ node hello.js
    <transaction hash>
    ```

1. Search the transaction hash on the [Devnet Explorer](https://devnet.thetangle.org/)
